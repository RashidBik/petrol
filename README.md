# نرم‌افزار ثبت و آنالیز داده‌های تانک تیل

وبسایت معرفی و دانلود نرم‌افزار تخصصی ثبت و آنالیز داده‌های تانک تیل

## ویژگی‌ها

- ثبت دقیق داده‌ها
- آنالیز پیشرفته
- داشبورد مدیریتی
- گزارش‌گیری حرفه‌ای

## تکنولوژی‌ها

- SvelteKit
- JavaScript
- HTML5
- CSS3

## نصب و راه‌اندازی

```bash
# نصب dependencies
npm install

# اجرای توسعه
npm run dev

# ساخت نسخه production
npm run build

# پیش‌نمایش نسخه build شده
npm run preview
